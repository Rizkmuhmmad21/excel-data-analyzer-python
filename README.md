Script Python untuk membaca data Excel, menghitung statistik dasar,
membuat grafik otomatis, dan mengekspor hasil ke Excel.

\\ 1. Fitur
- Input file Excel dari user
- Hitung mean, std, min, max
- Visualisasi line chart
- Export hasil ke Excel

\\ 2. Cara Pakai
1. Install dependency:
   pip install -r requirements.txt
2. Jalankan:
   python analyzer.py
3. Masukkan path file Excel saat diminta.

\\ 3. Contoh Output
- Output/Statistik Data.xlsx
- Output/Line Chart Data.png

- Analisis produksi
